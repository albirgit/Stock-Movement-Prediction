# Financial News Stock Movement Prediction

## Project Overview
This project predicts stock market movement (Up/Down) using financial news headlines.

## Dataset
Financial News for Stock Movement Prediction  
https://www.kaggle.com/datasets/aaron7sun/stocknews

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Preprocess text data
- Train a machine learning classification model
- Build an interactive Streamlit web app

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- NLTK
- Streamlit
